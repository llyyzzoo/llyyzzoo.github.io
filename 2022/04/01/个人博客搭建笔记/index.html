下载

下载git、Node.js、一个文本编辑器
教程：https://www.cnblogs.com/xueweisuoyong/p/11914045.html

检验下载

win+r再cmd
输入以下内容：

node -v
npm -v
git –version
hexo下载：npm install express -g
hexo -v

注册github

github账号注册：https://www.bilibili.com/read/cv5107169

创建仓库

仓库名应该为：

用户名.github.io

本地与github绑定

在任意位置右键git bash
生成ssh:

ssh-keygen -t rsa -C “邮箱地址”

.ssh生成路径：C:\Users\用户名.ssh
复制文件内容
打开GitHub_Settings_keys 页面，新建new SSH Key
粘贴文件内容
测定ssh是否绑定成功：

ssh -T git@github.com

如果一直出现 ssh: connect to host github.com port 22: Connection refused 的话具体看这个教程

创建

选好位置

hexo init #初始化hexo博客

如果报错SyntaxError: Unexpected token …建议更新一下node.js版本

hexo s #静态生成hexo页面

localhost:4000查看

配置

配置文件修改(_config.yml)

deploy:
  type: ‘git’
  repository: https://github.com/你的名字/仓库的名字.git
  branch: main

在根目录git bash
安装hexo-deployer-git自动部署发布工具:

npm install hexo-deployer-git –save

hexo g生成
hexo d部署
hexo s查看

主题

主题库

git clone 主题地址

_config.yml配置文件修改主题
站点配置文件_config.yml修改基本的站点信息

绑定域名

登录到阿里云，进入管理控制台的域名列表，找到你的个性化域名，进入解析包括添加三条解析记录

185.199.108.153
185.199.109.153
185.199.110.153
185.199.111.153

是GitHub的地址，你也可以ping你的 http://xxxx.github.io 的ip地址，填入进去。第三个记录类型是CNAME，CNAME的记录值是：你的用户名.http://github.io 这里千万别弄错了。
第二步，登录GitHub，进入之前创建的仓库，点击settings，设置Custom domain，输入你的域名，点击save保存。
第三步，进入本地博客文件夹 ，进入blog/source目录下，创建一个记事本文件，输入你的域名，对，只要写进你自己的域名即可。建议，不要带有www。保存，命名为CNAME ，注意保存成所有文件而不是txt文件。

常用的Hexo 命令

npm install hexo -g #安装Hexo
npm update hexo -g #升级
hexo init #初始化博客

命令简写
hexo n “我的博客” == hexo new “我的博客” #新建文章
hexo g == hexo generate #生成
hexo s == hexo server #启动服务预览
hexo d == hexo deploy #部署
