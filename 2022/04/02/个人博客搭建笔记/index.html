个人博客搭建笔记
2022-04-02

操作步骤

下载

下载git、Node.js、一个文本编辑器
教程：https://www.cnblogs.com/xueweisuoyong/p/11914045.html

检验下载

win+r再cmd
输入以下内容：
node -v
npm -v
git –version
hexo下载：npm install express -g
hexo -v

注册github

github账号注册：https://www.bilibili.com/read/cv5107169

创建仓库

本地与github绑定

在任意位置右键git bash
生成ssh:ssh-keygen -t rsa -C “邮箱地址”
.ssh生成路径：C:\Users\你他娘的用户名.ssh
复制
测定ssh是否绑定成功：ssh -T git@github.com

如果一直出现ssh: connect to host github.com port 22: Connection refused的话具体看这个教程https://www.cnblogs.com/Archer314/p/14641310.html

创建

选好位置
hexo init #初始化hexo博客
如果报错SyntaxError: Unexpected token …
建议更新一下node.js版本
hexo s #静态生成hexo页面

配置

配置文件修改(_config.yml)

在根目录git bash
安装hexo-deployer-git自动部署发布工具:
npm install hexo-deployer-git –save

hexo g生成
hexo d部署
hexo s查看
